How to Host Your Static Website on GitHub Pages for Free!

Posted on June 28, 2020 [Read count: ]
Why pay for the static hosting if you can get it for free? Yup! If you didn't know, let me tell you that GitHub does provide Static Website Hosting for free; That too, with a trusted SSL Certificate!

What is GitHub Pages?

GitHub Pages is a static site hosting service by GitHub that takes HTML, CSS, and JavaScript files straight from a repository on GitHub, optionally runs the files through a build process, and publishes a website.

Step 2. Creating a GitHub Repository:

Now, We need to create a GitHub Repository. So that, we can upload our code to that remote repository. You can do that by following these steps:

1. Go to https://github.com and login to your account. If you don't have an account, Please create one.

2. Click on the "Plus(+)" sign. (Should be in the upper right corner somewhere) and select "New Repository"

3. Here, just enter Repository Name and leave other options unchanged. Then, Click on "Create Repository"

Step 3. Setting Up Your Local and Remote Git Repositories:

Now, head to the folder we created in the first step (github-pages-tutorial) and open Git Bash (Assuming that you have it installed. If not, you can download it here.) and paste the following commands one by one:

$ git init
$ git add .
$ git commit -m "Initial Commit"
$ git remote add origin <link_to_your_remote_repo>
$ git push -u origin master

After this, you can go and check the GitHub Repo. You'll see the "index.html" file we created earlier is uploaded here. That ends the step 3!

Step 4. Setting up GitHub Pages:

This this final step, we'll see how to actually set up GitHub Pages to host your site:

1.Open GitHub Repository we just created, and click on "Settings". (Should be the last option in the upper navigatin panel.)

2. Now, scroll down until you see the heading "GitHub Pages".

3. In the source section select "master branch" option. After clicking, the page might refresh the page automatically.

4. You'll see, Your site is published at:
https://your_user_name.github.io/your_repo_name.

You can open the address given and see your site online!
