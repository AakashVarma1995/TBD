YouTube Channels Every Developer Should Follow in 2020

Posted on June 14, 2020 [Read count: ]
YouTube has become an ultimate source of Tutorials for the Programmers in the past few years. In fact, video tutorials can be the best source material if you're getting started with the a technology. So here, I've made a list of some youtube channels which you should be watching to learn new tech in 2020.

1.Traversy Media

Brad, the tutor at the Traversy Media, is a lot famous for his crash-courses. When I have to deliver something in the technology I'm totally unaware of, I always watch brad's crash courses before getting started!

Link : Traversy Media

2. codebasics

Dhawal Patel, having more than 15 years of working as a software engineer; has also worked with Nvidia Graphics and Bloomberg LP. Also, he's a great tutor.

Link : codebasics

3. Academind

Whether you want to have look at Angular Tutorials or Guides, Vue.js, other Frontend Development Content or Data Science Topics or anything else - you're probably right!

Link : Academind

4. Programming with Mosh

This channel is a holy grail of one of the best tutorials. Mosh teaches about React, React - Native, Redux, Data Structures, Design Patterns, Python, and many more technologies.

Link : Programming with Mosh

5. Design Course

Gary has created close to 100 courses from graphic design to advanced frontend development. He has also worked with Envato Network's TutsPlus.com, DigitalTutors.com, Pluralsight.com, LinkedIn Learning & Lynda.com.

Link : Design Course

6. Edureka!

Edureka! a live & interactive e-learning platform with the mission of making learning accessible to everyone. They offer instructor-led courses, along with 24/7 on-demand support to achieve highest course completion rates in the industry! Their real-life projects, 24*7 Support, Personal Learning Managers ensure that your learning goals are met!

Link : Edureka!

7. freeCodeCamp.org

We're an open source community of busy people who learn to code and build projects for nonprofits.

Link : freeCodeCamp.org

8. Codevolution

Has tutorials on the latest tech in web development like MEAN and MERN stack. Also, this channel has some other tutorials like Chrome Extension Development, How to use VS Code, VUE.js and NPM.

Link : Codevolution

9. Web Dev Simplified

Web Dev Simplified is all about teaching web development skills and techniques in an efficient and practical manner. If you are just getting started in web development Web Dev Simplified has all the tools you need to learn the newest and most popular technologies to convert you from a no stack to full stack developer. Web Dev Simplified also deep dives into advanced topics using the latest best practices for you seasoned web developers.

Link : Web Dev Simplified
