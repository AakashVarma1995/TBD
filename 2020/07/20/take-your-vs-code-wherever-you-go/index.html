Take your VS Code wherever you go! Here's how...

Posted on July 20, 2020 [Read count: ]
Every full-stack developer will relate to this issue! we use multiple machines for multiple projects. Though VS Code is a very powerful and versatile tool for developers. But, it lacks one fundamental thing... i.e an option to save user preferences and reflect them across all the devices. I get really irritated configuring all my comfy settings every single time on every single machine. Well, not anymore...

How? ---> Code-Server!

Code-Server is an open-source solution by [Coder]. It takes the open source core of VS Code and allows you to run it in the browser by hosting it onto a server. However, it is not entirely equivalent to Microsoft's VS Code.

Highlights:

  • Code on your Chromebook, tablet, and laptop with a consistent development environment.
  • Develop on a Linux machine and pick up from any device with a web browser.
  • Take advantage of large cloud servers to speed up tests, compilations, downloads, and more.
  • Preserve battery life when you're on the go as all intensive tasks runs on your server.
  • Make use of a spare computer you have lying around and turn it into a full development environment.

How To?

To setup code-server on the cloud, you'll need a cloud VM instance, or you can also set it up on your local machine. [CDR] provides a ready-to-use script to install code-server for Linux, macOS and FreeBSD.

Process :

Copy and paste the following command in the terminal to print the install steps:

$ curl -fsSL https://code-server.dev/install.sh | sh -s -- --dry-run

Now, you can run the installation script

$ curl -fsSL https://code-server.dev/install.sh | sh

The above script will download the respective installation package for your operating system and will try to install it. After the installation completes, you can start the code-server by running the following command:

$ code-server

Now, you can open your browser and start usign VS Code by typing http://your_host_ip:8080 in your browser!

The Takeaway...

code-server is based on the open-source version of VS Code. Though it inherits most of the features, there still are some compatibility issues mostly with extensions. You can read more about code-server here. If you liked the the post, please comment and share... Thank you!
